HTMLParser

Overview

A personal project with the purpose of parsing DOM objects from a given string. It analyzes the document's children, along with each child's children until the end to construct the complete tree that is ready for user interaction.

Highlights

  • Parse HTML, XML (RSS), supports MATH documents.
  • Get children by ID, classes and/or tags.
  • Able to capture CDATA sections.

Technologies / Languages
