Java Development

HTMLParser

Overview

A personal project with the purpose of parsing DOM objects from a given string. It analyzes the document's children, along with each child's children until the end to construct the complete tree that is ready for user interaction.

Highlights

  • Parse HTML, XML (RSS), supports MATH documents.
  • Get children by ID, classes and/or tags.
  • Able to capture CDATA sections.

Technologies / Languages

  • Pure Java

Android Development

MUST Student (Unofficial)

Overview

A personal application project that facilitates browsing the students zone on Misr University for Science and Technology's website for students. It is still currently under early development. The application provides a range of useful features not actually present in the current official MUST portal application and is in no way associated with the university in any legal or financial way.

Highlights (Current or planned)

  • Browse the students zone offline
  • Add multiple accounts and track their data. (Must be authenticated!)
  • Register courses from the application

Technologies / Languages

  • Java
  • Android SDK
  • Firebase Job Dispatcher
  • Firebase Analytics
  • AdMob (Donation)
  • HTMLParser (Own project)

Feed Recess

Overview

The graduation project developed for the Udacity's Android Developer Nanodegree course by Google. The main point of the application is to allow users to fetch news and articles from their favorite RSS feeds, and display them in various ways such as in a homescreen widget.

Highlights

  • Parse <rss> documents (<feed> documents not supported [yet])
  • Fetch data automatically in the background
  • Aggregate feeds into a single multi-feed
  • Separate in-app view from the widget view (in terms of aggregated view)

Technologies / Languages

  • Java
  • Android SDK
  • Firebase Job Dispatcher
  • Firebase Analytics
  • HTMLParser (Own project)

CGR Toolkit

Overview

A small application to help in Chaos Game Representation (CGR) encoding and decoding, along with integer encoding and decoding algorithms. The application allows viewing the steps for both encoding and decoding for each iteration, and also plotting graphs.

Highlights

  • Encode sequences with normal CGR and integer CGR.
  • Plot graphs for both techniques.

Technologies / Languages

  • Java
  • Android SDK, no plugins.
